# Extension Icon

Please add an icon.png file (512x512 pixels) to this directory.

The icon should represent the Claude Plugin Manager extension.

Suggested icon design:
- A puzzle piece or plugin symbol
- Combined with the Claude/Anthropic branding colors
- Clean, modern, and minimal design
- 512x512 pixels, PNG format

You can create the icon using:
- Figma
- Adobe Illustrator
- Canva
- Or any other design tool

Once created, save it as `icon.png` in this directory.
